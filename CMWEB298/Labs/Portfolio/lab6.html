Media Queries

What are Media Queries?

Media query is a CSS technique introduced in CSS3. It allows CSS styles to be applied conditionally depending on certain parameters. These parameters include media types, such as screen or print, and can target different characteristics of the user's device. The include orientation of the device, the screen width or height, whether the device supports hovering, and many more. With media queries, developers can make very complex CSS rules that make sure the viewing experience is the same for every possible device.

How to use Media Queries in CSS

Media queries can be applied a few different ways. One way is to put a media query in the head of an HTML page. By doing this, we can link certain stylesheets depending on what size the screen is. By using something like <link rel="stylesheet" href="small.css" media="(max-width: 600px)">, small screens under 600 pixels wide will get the "small.css" file applied.
For example, for devices over 1024 pixels wide, the picture will be the CSS3 logo. It will change to the HTML5 logo for devices under 1024 pixels wide.
This is a prime example of how media queries can really spruce up a webpage, but it is only a small trick in a sea of possibilities for media queries.

Using Media Queries for Printing

Another thing media queries are great for is printing. When web pages are printed, some aspects of the web page need to be changed. This is because web pages tend to have a lot of color and some elements are not visible on physical paper.
When there is a link on a webpage, users can only see that link when they click it to go visit the linked page. This is an issue for printed webpages. To fix it, we can use a media query for printed pages and change up the style for the printed result.
