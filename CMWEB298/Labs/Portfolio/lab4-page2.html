CSS Grid

The main content of this page is laid out using CSS Grid, a responsive CSS layout module. Each of these paragraphs have their own column and row assigned. When the window is collapsed or enlarged, the grid will respond accordingly, decreasing or increasing the number of rows and columns.

Media Queries

Media queries are great for responsive web pages. This technique allows websites to change depending on the width of the user's window. Setting CSS inside of a media query like this @media only screen and (max-width: 600px) makes the CSS inside only active if the window is smaller than 600 pixels.

Media queries work great with CSS grid as this is how the number of columns or rows will change.

Animations and Keyframes

CSS has a slew of different animations that can be used to transform elements. Keyframes is a CSS rule used to specify the animation code. Although quirky and a bit frustrating to use, keyframe animations can really spice up a web page. The background to this website is actually using keyframes. I have a background gradient set and a keyframe called "gradient" is moving the background around. This is just a simple keyframe animation and it still adds a cool effect.
