Accessible Form Example

This accessible form looks like a basic form, but there are certain hidden attributes that make it more accessible than other web pages. These hidden attributes are called ARIA (Accessible Rich Internet Applications). Although hidden to our eyes, screen readers can really benefit from the use of ARIA. In this form, any input that is required has an "aria-required='true'" attribute so that screen readers can see that input is required. When the user moves from an input without filling it out, the screen reader makes sure to tell the user that input is required before moving on.
