Question 1:
Tell me how to set the position of a background image on my web page using css and give me a couple of different sites that explain it.
Answer:
Using the background-position property, you can position a background image using one or two directions (e.g. left top, right bottom, center left, etc.). If no second direction is specified, it will default the second position to center. The values can also be percentages or specific lengths using (x, y) for the positions unless specified.
URL:
W3Schools, MDN Web Docs
Search Site Used:
Google
Criteria Used for Search:
"how to position a background image in css"
Rational:
Both W3Schools and MDN Web Docs are very reputable sites for HTML and CSS information. MDN is run by Mozilla and has information for HTML, CSS, JS, HTTP, and more. W3Schools covers even more coding languages than MDN and provides information for every single CSS and HTML element.
Question 2:
What was the average wheat yield per acre (in bushels) in the 1800's? What is it now?
Answer:
I found from people.vcu.edu that from 1880-1884, the average wheat yield per acre was 13.3 bushels. I also found that the national average wheat yield per acre was 50 bushels in 2020.
URL:
People.vcu.edu, PennState Extension
Search Site Used:
Google
Criteria Used for Search:
"average wheat yield per acre in the 1800's" & "average wheat yield per acre now"
Rational:
People.vcu.edu seems reputable because their sources are cited and the site looks like it is used for educational purposes and made by a professor. PennState Extension seems reputable because they list sources and are a reputable educational site.
Question 3:
American Colonists used a paint made out of a common beverage to paint their homes; what common beverage was a primary ingredient in this paint?
Answer:
American colonists used milk to paint furniture, interior walls, and woodwork. It was a mixture of milk protein, quicklime, and earth pigments.
URL:
Sioux City Journal
Search Site Used:
Google
Criteria Used for Search:
"american colonists paint mixtures"
Rational:
The Sioux City Journal has lots of articles and have reputable authors. I also confirmed this information on several other sites.
