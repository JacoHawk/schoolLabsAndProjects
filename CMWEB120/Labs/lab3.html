W3Schools and MDN Web Docs are incredibly helpful resources for all things programming. Whether you need help with front-end or back-end coding, they both have tons of info and tutorials!
YouTube is also an excellent resource for beginner coders. There are tons of videos and series teaching certain languages or helpful tips and tricks!

The three different CSS attachment methods are inline, embedded, and external.
Inline CSS is placed within the opening tag of an element using style="". It's best to use inline css to ensure all the content of the page is still easy to understand without an external CSS document.
Embedded CSS is placed within the header of the HTML document between a "style" tag. Embedded CSS has similar use cases ass inline, but neither should be used very often. When there is a large amount of CSS to be added to a page, it's alwasy best to use an external CSS document.
External CSS doucments are linked in the header and should be used for sites that have a lot of styling. While the other methods are easier, when creating large pages, they can be a real headache to use. External CSS documents allow for the use of ID's and classes which makes the design process much faster as it is easier and quicker to change styles around.
